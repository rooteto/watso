== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/watso-basic-chat/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to 'Settings > Watso Chat' to configure the options.
4. Add your WhatsApp number(s) and customize the appearance.
5. Save settings and the chat button will appear on your website.

**Quick Setup:**
1. Navigate to Settings > Watso Chat
2. Enable the plugin in General Settings
3. Add your WhatsApp number in Numbers & Titles tab
4. Customize appearance in Appearance & Colors tab
5. Save changes

== Frequently Asked Questions (Watso FAQ) ==

= Is this plugin completely free? =

Yes, Watso is completely free with no premium versions or paid features.

= Can I use multiple WhatsApp numbers? =

Yes! You can add unlimited WhatsApp numbers. With one number, users are redirected directly. With multiple numbers, a dropdown menu appears.

= Does it work with caching plugins? =

Yes, the plugin is designed to work with all major caching plugins including WP Super Cache, LiteSpeed Cache, and WP Fastest Cache.

= Can I customize the button appearance? =

Absolutely! You can change colors, upload custom icons, adjust border radius, and choose from 4 different positions.

= Does it support dark mode? =

Yes, the plugin automatically detects dark mode and you can upload separate icons for dark themes.

= Can I schedule when the button appears? =

Yes, you can set working hours for each day of the week and add holiday dates when the button should be hidden.

= Is it mobile responsive? =

Yes, the plugin is fully responsive and you can choose whether to show the button on mobile devices.

= Does it track analytics? =

Yes, it supports Google Analytics, Universal Analytics, and Facebook Meta tracking with customizable UTM parameters.

= Can I translate the plugin? =

Yes, the plugin is translation ready and includes POT files for easy translation.

= Does it affect site performance? =

No, the plugin is lightweight and optimized for performance with minimal impact on page load times.

== Privacy Policy ==

Watso WhatsApp Chat plugin does not collect any personal data from your website visitors. The plugin only:

* Stores configuration settings in your WordPress database
* May send analytics data to Google Analytics or Facebook Meta if you enable tracking features
* Does not store or transmit any visitor information to our servers

When visitors click the WhatsApp button, they are redirected to WhatsApp's official service where WhatsApp's privacy policy applies.
